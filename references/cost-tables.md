## 如何估算成本

1. **统计 API 调用频率** - 查看代码中的调用次数和频率
2. **确定使用量** - 估算每月/每天的调用次数
3. **应用定价** - 使用上表的价格计算
4. **考虑免费层** - 大多数 API 都有免费额度

## 成本优化建议

- 使用缓存减少重复调用
- 批量请求而非单个请求
- 监控 API 使用量
- 设置预算告警
- 考虑自托管替代方案
